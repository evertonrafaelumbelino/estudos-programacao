História da Web:

(Voltar)

O primeiro computador do mundo foi criado em 1944, sendo ele chamado de Mark I. Ele era um dispositivo extremamente grande, mas que foi essencial para a evolução da tecnologia, e para que pudessemos chegar no patamar em que estamos.

A tão conhecida Internet foi criada no período da Guerra Fria1; onde basicamente foi feita com o objetivo de passar informações e se comunicar entre diferentes locais no exército. E conforme o tempo ia passando, ela ia se desenvolvendo e se aprimorando cada vez mais.

Além disso, podemos citar alguns vários nomes que foram de grande importância para o desenvolvimento e crescimento da tecnologia, como:

  • Grace Murray Hopper2
  • Joseph Carl Robnett Licklider3
  • Robert E. Kahn4
  • Tim Berners-Lee5
  • Marc Andreessen6
  • Bill Gates7
  • Steve Jobs8

Clients:

(Voltar)

Clients ou clientes, são basicamentes os usuários e pessoas que usam a internet e buscam informações dentro dela. Eles são capazes de acessá-la a partir de dispositivos como celulares, tablets, PCs, entre outros que tenham acesso a internet.

Os clientes fazem um "pedido" (ou requisição) ao navegador, onde esses vão pegar essa requisição e buscá-la no servidor por meio da internet, e retornar a resposta encontrada.

Servers:

(Voltar)

Servers ou servidores são a parte que não é visível ao usuário, mas é a que faz com que as coisas funcionem por "de trás dos panos".

São eles que pegam a requisição enviada pelo navegador, e a buscam na prática em bancos de dados9 ou em outros locais para encontrar a resposta e retorná-la ao cliente.

Linguagem de programação

(Voltar)

Uma linguagem de programação, assim como o nome já diz, é simplesmente a "lingua" que os programadores usam para se comunicar com os computadores, executar comandos e resolver tarefas neles.

Existem dois tipos principais de linguagens de programação: as Client-side (operam no lado do cliente) e as Server-side (operam no lado do servidor). Onde podemos citar alguns exemplos como:

JavaScript (Node.JS), C#, Java e PHP para o lado do servidor; e JavaScript novamente, só que para o lado do cliente.

HTML:

(Voltar)

O HTML diferente do que muitos acreditam, não é uma linguagem de programação, mas sim uma linguagem de marcação de texto. Sendo basicamente um tipo de arquivo que contém elementos e tags10 usadas para formatar os conteúdos no navegador de formas diferentes.
